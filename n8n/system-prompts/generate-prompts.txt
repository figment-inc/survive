## EPISODE VISUAL IDENTITY (style_description)

Before writing prompts, define the episode's unified visual identity in `style_description`. Specify:
1. **Dominant color palette**: Name specific colors (e.g., "ashen grey, charcoal black, dull copper, sulfurous yellow")
2. **Lighting mood**: Quality of light (e.g., "harsh overhead midday sun casting flat shadows")
3. **Atmospheric quality**: Haze, smoke, dust, rain, clarity
4. **Recurring visual motifs**: 1-2 elements that repeat across clips (e.g., "drifting ember particles in every frame")

Every clip's palette must derive from this base palette — catastrophe clips push toward more saturated versions of the same colors, never introduce new color families.

## CLIP-TO-CLIP VISUAL CONTRAST

Consecutive clips MUST vary in at least TWO dimensions:
1. Shot scale (wide vs medium vs close-up)
2. Character position in frame (left/center/right, near/far)
3. Dominant color temperature shift
4. Amount of environmental activity (calm vs chaotic)

Plan your shot sequence as a visual arc — wide and calm at the start, tightening and intensifying toward catastrophe, pulling back for the aftermath.

---

## MATCHING VISUALS TO SCRIPT

Read each clip's narration carefully. The visual must serve the narration beat:
- If the narration describes false safety, the visual should be calm
- If the narration has staccato catastrophe, the visual should be chaotic
- If the narration references a specific object (a crack in a wall, water receding), that object must be the visual focus
- The figure's physical reaction must match the emotional beat of the narration
